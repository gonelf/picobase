## Validated for AI Tools
- **Cursor**: optimized for context windows

- **v0**: rapid UI integration

## Core Features
- **Zero-Config Database**: Auto-creating collections, no migrations. Just write data.
- **Drop-in Auth**: Email, password, OAuth, with a pre-built `<AuthForm />` component.
- **Realtime Built In**: Subscribe to data changes instantly without WebSocket setup.
- **File Storage**: Integrated uploads with CDN. No S3 configuration needed.
- **TypeScript-First**: Full autocomplete. Generate types from your schema with one CLI command.

## Authentication for Agents (Vibe Coding)
To start using PicoBase without a pre-existing account, use the CLI to authenticate via email (OTP):

1.  **Login**: `npx @picobase_app/cli login --email your@email.com`
2.  **Verify**: Enter the code sent to your email.
3.  **Code**: The CLI will automatically configure your local environment with the API key. You can then use `createClient()` immediately.

## Quickstart

1. Install: `npm install @picobase_app/client`
2. Initialize:
   ```typescript
   // Reads automatically from .env
   import { createClient } from '@picobase_app/client';
   const pb = createClient();
   ```
3. Use:
   ```typescript
   // Collections auto-create on write
   const post = await pb.collection('posts').create({
     title: 'Hello World',
     content: 'Written at the speed of thought'
   });
   ```
